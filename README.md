# 🫀 Heart Disease Prediction using Machine Learning

##  Project Overview
This project predicts whether a person is at risk of heart disease using Machine Learning techniques and medical health data from the Heart Disease UCI Dataset.

---

##  Objective
Build a classification model to identify heart disease risk based on patient medical information.

---

##  Dataset
- Heart Disease UCI Dataset
- Source: Kaggle

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

##  Project Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Encoding
5. Train-Test Split
6. Feature Scaling
7. Model Training
8. Model Evaluation
9. Feature Importance Analysis

---

##  Machine Learning Model
